Popcorn

Description

Popcorn is a specific type of corn, that pops when heated up with some oil. As you might expect, this recipe is very simple.

Ingredients (per 100g of popcorn)

  • 100g "raw" popcorn
  • 50g oil
  • other ingredients you might want to add: salt/sugar, caramel, other spices...

Steps

  1. Heat up the oil on medium-low heat in a pot with a lid
  2. Add the raw popcorn, if you haven't done so put on the lid
  3. The popcorn will start to audibly pop (keep the lid on to make sure that it doesn't jump out). Every now and then, shake up the pot a bit to prevent the corn from burning.
  4. If almost all the corn popped, it is done. You probably won't get all of the corn to pop.
  5. Now you can add salt and other optional ingredients.

Enjoy!
